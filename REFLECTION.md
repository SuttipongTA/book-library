ฝั่ง Backend ผมพอจะถนัดและคุ้นเคยกับ Node.js/Express อยู่แล้ว แต่ส่วนที่ยากที่สุดสำหรับผมคือฝั่ง Frontend เพราะไม่ถนัดเท่าฝั่ง Backend และตัวผมเองก็ไม่ได้เขียนฝั่ง frontend มานานผมเลยให้ AI ช่วยเจนโค้ดโครงส่วนใหญ่ของ Frontend ให้ก่อน แล้วอ่านไล่ทำความเข้าใจทีละไฟล์ แล้วไปอ่านวิธีการทำงานของฟังก์ชันต่าง ๆ จนพอจะอธิบาย flow การทำงานได้ เช่น การเก็บ token ใน AuthContext การแนบ token อัตโนมัติผ่าน axios interceptor, และการ debounce ค้นหาหนังสือ เป็นต้น แล้วค่อยเริ่มทำส่วนอื่น ๆ ต่อ แต่ก็ยังมีจุดที่ยังไม่แม่นอยู่ในส่วนพวก useMemo use ต่าง ๆ ที่ยังใช้ได้ไม่คล่องนัก
